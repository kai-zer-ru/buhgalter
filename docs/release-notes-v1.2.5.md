# Release notes v1.2.5

## In-app уведомления (toast)

Краткие сообщения об успехе, ошибке и предупреждении теперь показываются **внизу справа** (на узком экране — по центру снизу).

- Сохранение, удаление, ошибки API и валидация при отправке формы — через toast
- Не затрагивает уведомления в **Telegram** и **MAX** — это по-прежнему отдельная настройка

Подробности для разработчиков: [ui-toast.md](ui-toast.md).
