## Конфигурация

### Поддержка `.env` файла

Приложение читает `.env` файл при старте. Это удобно для задания переменных окружения без правки системных unit-файлов или Docker Compose.

**Где разместить:**
- **Бинарник** — `.env` в каталоге, из которого запускаете `./buhgalter` (обычно рядом с бинарником). Если запускаете из другого места, укажите путь через переменную `BUHGALTER_ENV_FILE`
- **Docker Compose** — `.env` рядом с `docker-compose.yml`

**Основной сценарий:** задать `BUHGALTER_ALLOWED_HOSTS` для доступа с других устройств в LAN или с удалённого сервера (без reverse proxy). localhost/127.0.0.1 доступны всегда.

```env
BUHGALTER_ALLOWED_HOSTS=["192.168.1.100","example.com"]
```

Подробнее — [README.md](../README.md#переменные-окружения).

## Интерфейс

### Счета — в верхнем меню

Пункт **«Счета»** перенесён из вкладок Настроек в верхнее меню навигации. Теперь он доступен на всех страницах, а не только внутри Настроек.

- Страница `/accounts` — полноценный список счетов с балансами (раньше редиректила в Настройки)
- Кнопка «Счета» подсвечивается, когда вы находитесь на странице счёта

### Курсор-указатель на кнопках

При наведении на все кнопки и кликабельные элементы теперь отображается указатель (рука) вместо стрелки.

### Мелочи

- Кнопка «Новый счёт» на главной теперь одной высоты с кнопкой «+»
- Расстояние между «Админка» и именем пользователя в шапке увеличено для лучшей читаемости
