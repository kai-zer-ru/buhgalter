## Единые форматы даты и времени

Во всём интерфейсе даты и время выводятся в одном стиле:

| Что | Формат | Пример |
|-----|--------|--------|
| Только дата | `дд.мм.гггг` | `31.12.2026` |
| Дата и время | `дд.мм.гггг чч:мм:сс` | `31.12.2026 12:00:00` |
| Дата и время в списках операций и уведомлениях | без секунд | `31.12.2026 12:00` |

Затронуты:

- списки операций (главная, «Все операции», страница счёта), долги, кредиты, периодические операции
- фильтры и пикеры даты
- исходящие уведомления (Telegram, MAX): плейсхолдеры `{date}`, `{requested_at}` — без секунд; `{due_date}`, `{payment_date}` — только дата
- **админка:** диагностика (в том числе **время сборки**), список бекапов — с секундами

Хранение в API и базе не менялось (`yyyy-MM-dd HH:mm:ss`, RFC3339).

Подробнее для разработчиков — [date-time-display.md](date-time-display.md).

## Админка

### Диагностика

Поле **build_time** (время сборки) показывается в человекочитаемом формате `дд.мм.гггг чч:мм:сс`, как и даты в разделе бекапов.

### Сброс пароля из уведомления

При переходе по ссылке из уведомления (`/admin/users?reset=…`) модальное окно сброса пароля **закрывается по кнопке «Отмена»**; параметр `reset` убирается из адресной строки, окно не открывается повторно.

### Ссылки «Поддержать проект» и «Репозиторий»

Кнопки снова видны:

- во вкладке **Настройки → Администрирование → Система**
- на маршрутах `/admin` и вложенных страницах админки

Компонент `AdminSupportLinks` — единый блок ссылок для обоих мест.

## Хлебные крошки

На всех страницах разделов и вложенных экранах сверху отображается цепочка **Главная → … → текущий экран**:

- **Счета**, **Долги**, **Кредиты**, **Операции**, **Статистика**, **Периодические операции** — как раньше, плюс список счетов (`/accounts`)
- **Настройки** — «Главная / Настройки»; при смене вкладки (Пароль, Категории, Импорт и т.д.) добавляется имя вкладки
- **Админка** (`/admin/*`) — «Главная / Настройки / Админка» и при необходимости вкладка (Пользователи, Бекапы, Диагностика)

Клик по «Главная» возвращает на дашборд. Подробнее — [ui-navigation.md](ui-navigation.md).
