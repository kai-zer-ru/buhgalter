## Производительность

Главная, списки операций, страницы счетов и статистика должны открываться заметно быстрее, особенно при большом числе операций.

- Баланс счёта хранится в базе и обновляется при изменении операций — больше не пересчитывается с нуля на каждый запрос
- Дашборд и прогноз баланса считаются пакетными запросами по всем счетам сразу
- Сервер кеширует GET-ответы в памяти (справочники — 5 мин, остальное — 1 мин); при любой правке данных кеш пользователя сбрасывается
- Веб-клиент при старте загружает справочники одним запросом `GET /ui/meta` (счета, банки, категории, должники, кредиты)

Подробнее для разработчиков — [ui-api-cache.md](ui-api-cache.md).

## Операции (доход, расход, перевод)

На **главной**, в **«Все операции»** и на **странице счёта** — три кнопки-иконки:

| Кнопка | Действие |
|--------|----------|
| **+** (доход) | Новая операция дохода |
| **−** (расход) | Новая операция расхода |
| **⇄** (перевод) | Перевод между счетами |

Подсказка при наведении — полное название.

- На **странице счёта** (десктоп): кнопки в шапке карточки рядом с меню «⋯» (редактирование счёта, основной, архив, удаление)
- На **странице счёта** (мобильные): кнопки дохода, расхода и перевода — в меню «⋯»
- В **«Все операции»** добавлена отдельная кнопка **Перевод** (раньше перевод был только через общую «Операция»)

**Форма новой операции:** переключатель «Доход / Расход» убран — тип уже выбран кнопкой; в заголовке окна отображается «Доход» или «Расход». При **редактировании** существующей операции тип по-прежнему показан текстом и не меняется.

Подробнее — [transactions-display.md](transactions-display.md), [ui-row-actions.md](ui-row-actions.md).

## Запланированные операции (future)

Операции с датой в будущем активируются **в фоне** (раз в минуту и при старте сервера), а не при каждом открытии списка или главной. Допустимая задержка — до ~1 минуты после наступления даты.

## Статистика

- Блок **«По категориям»** разделён на **доходы** и **расходы**
- При фильтре по типу показывается только соответствующая секция
- Процент категории считается от суммы категорий **того же типа** (а не от общего оборота)
- Поле **«Разница»** в контекстной статистике и на `/stats` убрано — дублировало баланс

Подробнее — [ui-stats.md](ui-stats.md).

## Админка

- Баннер запросов сброса пароля загружает список **один раз** при открытии приложения, без постоянных фоновых запросов при навигации по страницам

## Исправления

- **Страница счёта:** клик по счёту-контрагенту в строке перевода корректно открывает страницу другого счёта — и для входящих («на …»), и для исходящих («с …») переводов
