# UI — пагинация

Единый компонент `TransactionPagination` (`web/src/lib/components/TransactionPagination.svelte`) для постраничной навигации в списках.

Связанные документы: [transactions-display.md](transactions-display.md), [ui-credits.md](ui-credits.md).

---

## Параметры

| Prop | Тип | Описание |
|------|-----|----------|
| `page` | `number` | Текущая страница (с 1) |
| `limit` | `number` | Записей на страницу |
| `total` | `number` | Общее число записей |
| `onchange` | `(page: number) => void` | Переход на другую страницу |
| `class` | `string` | Дополнительные классы обёртки (отступы, рамка) |

Блок пагинации **скрыт**, если все записи помещаются на одну страницу.

## Кнопки

Четыре действия (i18n: `transactions.pagination.*`):

| Кнопка | Ключ | Когда неактивна |
|--------|------|-----------------|
| В начало | `first` | `page <= 1` |
| Назад | `prev` | `page <= 1` |
| Вперёд | `next` | `page >= totalPages` |
| В конец | `last` | `page >= totalPages` |

Все четыре кнопки **всегда видны** (чтобы не прыгала вёрстка); недоступные — `disabled` (класс `btn-ghost`: пониженная непрозрачность, без hover). Между кнопками — подпись «Страница N из M».

## Где используется

| Экран | `limit` | Синхронизация `page` |
|-------|---------|----------------------|
| `/transactions` | 20 | URL `?page=` |
| `/accounts/[id]` (операции счёта) | 20 | URL `?page=` |
| `/credits/[id]` (группы графика) | 10 | локальный state |
| Форма создания кредита (превью графика) | 10 | локальный state |

При добавлении нового списка с пагинацией — использовать этот компонент, не дублировать разметку кнопок.
