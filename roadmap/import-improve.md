# Улучшение импорта

Планируется в **v1.6.0** ([ROADMAP](../ROADMAP.md#v160)).

## Зачем

Сейчас импорт — в основном Cubux CSV/XLSX. Пользователи с выгрузками российских банков и других программ упираются в ручной маппинг «через Cubux» или ручной ввод.

Ориентир — **форматы и банки внутри РФ** (и универсальный CSV), не западный Open Banking.

## Scope (v1.6.0)

| Возможность | Суть |
|-------------|------|
| Универсальный CSV | UI «сопоставь колонки» (дата, сумма, описание, категория…) без жёсткого парсера на каждый банк |
| Популярные выгрузки РФ | По возможности 1–2 готовых профиля (какие — по запросам пользователей) |
| Dedup | Не создавать дубли при повторной загрузке той же выписки |
| Связь с правилами | После импорта — правила категорий и очередь «разобрать» |
