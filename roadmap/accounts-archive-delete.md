# Архивация и удаление счётов

Реализовано в v1.3.0.

Документация: [docs/accounts-archive-delete.md](../docs/accounts-archive-delete.md).

## Кратко

- **Архив** и **мягкое удаление** (`status = archived` / `deleted`); операции сохраняются.
- **`cash` / `bank`:** при `balance > 0` — автоперевод остатка перед архивацией/удалением (`transfer_to_account_id`).
- **`credit_card`:** перевод не нужен; только при полном погашении (`balance >= credit_limit`).
- UI: диалог с выбором счёта-приёмника, счёт по умолчанию — основной; на удалённом счёте операции только «Повторить».
