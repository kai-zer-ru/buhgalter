# Поддержание баланса

**Реализовано.** Источник правды: [docs/balance-maintenance.md](../docs/balance-maintenance.md), UI — [docs/ui-balance-maintenance.md](../docs/ui-balance-maintenance.md).

## Концепция

Включено (только `bank`):

- При `current_balance < порога` — автоматический перевод до целевой суммы.
- Категория «Перевод», комментарий `Автопополнение`.
- Счёт списания — другой активный `bank`.

Выключено:

- Для `cash` и `credit_card`.
- При нехватке средств на источнике — автопополнение отключается, уведомление `auto_topup_disabled`.

## UI

- Главная и страница счёта: `Автопополнение: {имя счёта списания}` у bank-счетов с включённым автопополнением; при выключенном — строка скрыта.
- Меню счёта: пункт «Автопополнение» → диалог с порогом, целью, источником.
